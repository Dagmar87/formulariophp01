# Formulario PHP

Esse projeto de estudo consiste na criação de um formulario simples de contato utilizando as linguagens HTML, CSS e PHP.

## Linguagens e Tecnologias utilizadas

- PHP 8.1
- HTML
- CSS
- IDE Apache NetBeans
